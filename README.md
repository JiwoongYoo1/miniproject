
📋 Devel-rope
==================
항해99 6조 6기 - 미니 프로젝트!
맛있는 떡볶이집을 이미지로 소개하고 실시간채팅으로 내용을 공유하는 커뮤니티입니다!!
떡볶이 사진 이미지를 업로드할수있고 메인페이지에서 다른사람들이 올린 내용도 확인 할 수 있습니다.
확인하고싶은 사진을 클릭하면 다른사람들이 올려놓은 사진과 음식점의 이름 올린이의 소감을 알수 있습니다.
